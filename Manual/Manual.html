On the Subject of Non-Stop Debates

Anything I found, anything I noticed... I have to be ready to speak up about everything. Because this isn't just about me. Everyone's lives are on the line!

NOTE: THIS MODULE CONTAINS SMALL SPOILERS FOR THE SERIES

SEE APPENDIX CL4SS3S FOR A LIST OF ALL STUDENTS

  • 16 students were involved in a bomb incident which one of them committed, and they were chosen to find out who did it.
  • The 16 students will testify about the modules on that bomb. They don't have to be from this bomb.
  • The arguments can be found on this module. Every 10 seconds, the next student will give their argument.
  • The arrow on the right shows how far the argument is. When it reaches the end, it will loop to the begin.
  • From a total of 5 arguments, there is one that doesn't match the manual of the module they are discussing.
  • An inconsistency can be refuted by using evidence. They will be referred to as "Truth bullets".
  • When holding the cylinder for 2 second ticks, the bullet will turn from a "Truth Bullet", to a "Memory Bullet". For more information, see Appendix C0NS3NT.
  • Once you find an inconsistency, use the correct truth bullet by pressing on the cylinder, and refute by clicking on the argument.
  • To solve this module, refute 3 arguments by finding inconsistencies in the arguments of the other students.

Appendix C0NS3NT: Memory Bullets

If you hold the cylinder for 2 second ticks of the bomb, the bullet will turn into a "Memory Bullet". Instead of refuting, the bullet indicates an agreement to the statement.

Section A: When to use it

At least 2 of the following actions must apply to fire a Memory Bullet:

  • If all statements are truths.
  • If the bomb has more than 10 total widgets.
  • If two names of the same class* are present.
  • If a subject appears at least twice.

Note: You are required to shoot a Memory bullet if at least 2 of the actions apply, even if a different weak spot is inconsistent. See Section B for the exception to this rule. You can remove the Memory Bullet if you hold the cylinder for another 2 bomb ticks. (* See appendix CL4SS3S for a student record.)

Section B: Where to use it

To find out where to fire, follow the rules listed below. Perform the first action that applies:

  • If both Makoto Naegi and Kyoko Kirigiri are present, agree with Makoto
  • If both Hajime Hinata and Chiaki Nanami are present, agree with Hajime
  • If both Kaede Akamatsu and Suichi Saiharra are present, agree with Suichi.
  • If either Kyoko Kirigiri or Byakuya Togami are present, agree with them if the serial number contains a character that is also in the word "S34RCH"
  • If either Chihiro Fujisaki or Chiaki Nanami are present, agree with them if the serial number contains a character that is also in the word "V1R7U4L"
  • If either Makoto Naegi or Nagito Komaeda are present, agree with them if the serial number contains a character that is also in the word "H0P3""
  • If none of the above apply, agree with the first statement, except if it's from either Celestia Ludenberg or Kokichi Oma. In that case, agree with the next statement. (Liars...)

Exception:

If the statement you should fire a Memory Bullet at, is inconsistent, do not fire a Memory Bullet at it. Insted, fire a Truth Bullet at it.

Section C: Which to use

Shoot the bullet from the same module that best matches te statement. For example, use "Value of color" at Accumulation, or "Possible departure cities" when talking about departure cities at European Travel.
